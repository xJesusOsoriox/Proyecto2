Tabla de enrutamiento

Cada equipo conectado a una red IP necesita una tabla de direccionamiento, ya que permite la interconexión de todas las subredes que componen Internet.












Tabla de enrutamiento
Dirección de Red
Para buscar la ruta adecuada se trabaja con el dato de la siguiente columna.
Máscara de Red
Permite diferenciar la parte de la dirección destino que se refiere a red de la de host.
Puerta de acceso
Especifica la dirección IP del dispositivo al que se debe enviar el paquete de datos después de detectar una coincidencia con los valores anteriores.
Interfaz
Encargado de enviar el paquete de datos a la ruta asignada.
Métrica
Número máximo de saltos hasta alcanzar el destino.
